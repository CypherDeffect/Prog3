Como adotar um dispositivo UniFI por SSH

  1. Conecte o dispositivo em uma porta livre de seu Switch

  2. Verifique o IP que foi atribuido ao dispositivo
  3. Abra o Putty e entre com o IP do dispoitivo
  4. Entre com o seguinte comando no terminal
    5.   
                    set-inform http://ipcontroller:8080/inform
  5. O dispoitivo irá aparecer para adoção no Controller. Clique em apopt para adotar
